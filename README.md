# Machine Learning Internship Tasks

This repository contains machine learning projects completed as part of an internship.

## Tasks Included

### Task 1: Spam Mail Detector
- Classifies emails/messages as **Spam** or **Ham**
- Uses machine learning and text processing techniques
- Implemented using Python and Scikit-learn

### Task 2: House Price Prediction
- Predicts house prices based on numerical features
- Uses regression algorithms
- Implemented using Python and Scikit-learn

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
